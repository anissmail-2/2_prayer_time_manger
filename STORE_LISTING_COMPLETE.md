## Developer Account Setup

### Google Play Console

**Cost:** $25 (one-time fee)

**Requirements:**
- Google account
- Valid payment method
- Developer profile information
- Privacy policy URL (hosted publicly)

**Setup Steps:**

1. **Create Account**
   ```
   1. Visit: https://play.google.com/console/signup
   2. Sign in with Google account
   3. Pay $25 registration fee
   4. Agree to Developer Distribution Agreement
   5. Complete account details
   ```

2. **Developer Profile**
   ```
   - Developer name: [Your Name/Company]
   - Email address: [Contact email]
   - Phone number: [Required for verification]
   - Website: [Optional but recommended]
   ```

3. **Payment Profile**
   ```
   - Required if you plan to sell apps/in-app purchases
   - Can skip initially for free apps
   - Add later when needed
   ```

4. **Create App**
   ```
   1. Click "Create app"
   2. Enter app details:
      - Default language: English (United States)
      - App name: TaskFlow Pro
      - Type: App
   3. Agree to declarations
   4. Create app
   ```

5. **Dashboard Setup**
   ```
   Complete required sections:
   - App access (no restrictions)
   - Ads (No, app is ad-free)
   - Content rating (Everyone)
   - Target audience (13+)
   - News app (No)
   - COVID-19 contact tracing (No)
   - Data safety
   - App category
   - Store listing
   - Main store listing
   ```

### Apple Developer Program

**Cost:** $99/year (recurring)

**Requirements:**
- Apple ID
- Valid payment method
- Identity verification (government ID)
- D-U-N-S Number (for organizations)

**Setup Steps:**

1. **Enroll in Program**
   ```
   1. Visit: https://developer.apple.com/programs/enroll/
   2. Sign in with Apple ID
   3. Choose entity type:
      - Individual (personal)
      - Organization (company) - requires D-U-N-S
   4. Complete enrollment form
   5. Pay $99 annual fee
   6. Wait for approval (1-2 days for individual, 1-2 weeks for org)
   ```

2. **App Store Connect Account**
   ```
   1. Visit: https://appstoreconnect.apple.com/
   2. Sign in with developer Apple ID
   3. Accept agreements
   4. Complete tax and banking info (if selling)
   ```

3. **Create App**
   ```
   1. Click "My Apps" > "+" > "New App"
   2. Fill in details:
      - Platforms: iOS
      - Name: TaskFlow Pro
      - Primary Language: English (U.S.)
      - Bundle ID: com.awkati.taskflow
      - SKU: taskflow-pro-001
   3. Create app
   ```

4. **App Information**
   ```
   Complete sections:
   - General information
   - App Store information
   - Pricing and availability
   - App Privacy
   - TestFlight (for beta testing)
   ```

1. **Test thoroughly** - No crashes or major bugs

3. **Privacy policy** - Must be comprehensive and accessible

5. **Complete metadata** - Fill all required fields
6. **Quality screenshots** - Professional, clear, accurate
7. **Functional app** - All core features work

## After Submission

**Google Play:**
- Review time: 1-3 days (sometimes hours)
- Check "Dashboard" > "Publishing overview" for status
- Respond to any issues promptly
- Once approved, live within hours

**App Store:**
- Review time: 1-3 days average
- Check App Store Connect for status
- Use "Resolution Center" if rejected
- Once approved, can set release date or go live immediately
